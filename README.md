# DRF_With_Tests 


#### Install requirements

`pip install -r requirements.txt`

#### Run app 

- * Apply migrations 
    -
        * `python manage.py makemigrations`
        * `python manage.py migrate`

- * Run the server 
    -
        * `python manage.py runserver`
